# AI API 키 무료 발급 및 연동 가이드

패트릭님, 이미 제가 구현해 드린 코드가 작동하려면 **무료 API 키**가 필요합니다. 신용카드 등록 없이도 이메일 인증만으로 바로 발급받을 수 있는 가장 좋은 방법들을 정리해 드립니다.

---

## 1. Groq Cloud API 발급 (텍스트/LLM용)
포트폴리오 비서의 "두뇌" 역할을 합니다. 현재 가장 빠른 응답 속도를 제공하며, Llama 3B 모델을 무료로 사용할 수 있습니다.

1.  **[Groq Console](https://console.groq.com/keys)** 에 접속합니다.
2.  Google이나 GitHub 계정으로 로그인합니다.
3.  좌측 메뉴에서 **"API Keys"**를 클릭합니다.
4.  **"Create API Key"** 버튼을 눌러 키를 생성하고 복사합니다.
5.  `js/modules/ai-assistant.js` 파일의 `GROQ_API_KEY` 부분에 붙여넣습니다.

---

## 2. Hugging Face Access Token 발급 (이미지/음악용)
이미지를 그리고 음악을 만드는 기능을 담당합니다. 수만 개의 오픈소스 모델을 무료 API로 연계해 줍니다.

1.  **[Hugging Face Settings](https://huggingface.co/settings/tokens)** 에 접속합니다.
2.  계정이 없다면 가입 후 이메일 인증을 완료합니다.
3.  **"Create new token"** 버튼을 클릭합니다.
4.  Token type을 **"Read"**로 설정하고 이름을 `Portfolio-AI` 정도로 지어줍니다.
5.  생성된 토큰을 복사하여 `js/modules/ai-assistant.js` 파일의 `HF_TOKEN` 부분에 붙여넣습니다.

---

## 3. "키 없이" 무료로 하는 방법 (WebGPU / Transformers.js)
API 키를 관리하기 번거롭거나, 서버 비용 걱정 없이(평생 무료) 운영하고 싶을 때 사용하는 방식입니다.

-   **방식:** 모델 데이터를 서버가 아닌 사용자의 브라우저(GPU)에서 직접 내려받아 실행합니다.
-   **장점:** API 키가 필요 없고, 개인정보 보안에 탁월하며 서버 부하가 없습니다.
-   **단점:** 사용자가 페이지에 접속할 때 약 100MB ~ 2GB 사이의 모델 파일을 다운로드해야 하므로 초기 로딩이 길어질 수 있습니다.
-   **추천 모델:** `SmolLM-135M` (텍스트), `Xenova/distil-whisper` (음성인식) 등

---

## 4. 패트릭님을 위한 추천 전략
> [!TIP]
> **기업용 포트폴리오**라면 **Groq + Hugging Face API** 방식을 추천합니다!
