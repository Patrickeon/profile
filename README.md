8년 차 베테랑 개발자 **Patrick**의 전문성과 기술 여정을 담은 최첨단 AI 테마 포트폴리오 사이트입니다. 단순한 이력 나열을 넘어, 기술 간의 유기적 관계와 프로젝트 히스토리를 인터랙티브하게 시뮬레이션합니다.

## ✨ Key Features

### 🧠 Massive Tech Neural Graph
- 25개 이상의 기술 스택이 유기적으로 연결된 신경망 형태의 그래프.
- 각 노드 호버 시 관련 기술군이 하이라이트되는 고밀도 인터랙션 제공.

### 📜 3D Project Timeline
- **GSAP ScrollTrigger**를 이용한 가로 스크롤 타임라인.
- 2012년부터 2026년까지의 13개 핵심 프로젝트 수록.
- **Vanilla-tilt.js** 기반의 3D 입체 투명 카드 UI.

### ⚡ Serverless SMTP Terminal Chat
- 터미널 테마의 인터랙티브 컨택 폼.
- **EmailJS SDK**를 사용하여 백엔드 없이 안전한 실시간 이메일 발송.

### 🤖 Floating AI Assistant
- Patrick의 경력과 기술 스택에 대해 즉각 응답하는 플로팅 챗봇 위젯.

## 🛠️ Tech Stack

- **Core:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Animations:** GSAP, ScrollTrigger, tsParticles
- **Interactions:** Vanilla-tilt.js, SVG Neural Logic
- **Services:** EmailJS (SMTP Bridge)

## 🚀 Getting Started

1. **로컬 실행:**
   ```bash
   npx serve .
   ```
2. **필수 설정:**
   - `js/main.js`의 `emailjs.init("YOUR_PUBLIC_KEY")` 및 `emailjs.send(...)` 부분을 본인의 키로 교체하세요.
   - `assets/images/profile.jpg`에 본인의 사진을 추가하세요.
